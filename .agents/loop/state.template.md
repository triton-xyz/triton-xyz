# State Template

Use this file only as a reference shape for `.agents/loop/state.md`. Keep `.agents/loop/state.md` focused on the current project task, not on workflow documentation.

# Mission

- State the current project goal in one sentence.

# Constraints

- List the hard technical and validation constraints for the current task.

# Current Strategy

- Record the current working hypothesis, knowing it may be replaced.

# Evidence

- Add short factual bullets with dates, commands, failures, or observations worth keeping.

# Next Options

- Keep `1` to `5` concrete candidate next moves.

# Blockers

- Record only real blockers. Use `None.` when clear.
